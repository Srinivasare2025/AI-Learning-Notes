# What is AI?

Artificial Intelligence is the capability of machines to simulate human intelligence.

Examples:
- ChatGPT
- Microsoft Copilot
- Face recognition
- Recommendation systems

---

# What is Generative AI?

Generative AI creates new content:
- Text
- Images
- Code
- Audio

Examples:
- ChatGPT
- GitHub Copilot
- DALL-E

---

# What is AI Foundry?

Azure AI Foundry is Microsoft's platform to:
- Build AI applications
- Manage AI models
- Create AI agents
- Test prompts
- Deploy endpoints

---

# What is an AI Model?

AI Model is the brain of AI.

Examples:
- GPT-4
- Llama
- Claude

Models process prompts and generate responses.

---

# What is a Prompt?

A prompt is the instruction given to AI.

Example:
"Summarize this SharePoint document"

Better prompts produce better results.

---

# What is an Endpoint?

Endpoint is the URL/API through which applications communicate with AI models.

Applications send prompts to endpoints and receive responses.

---

# What is an AI Agent?

AI agent is an intelligent assistant that:
- Understands goals
- Makes decisions
- Performs tasks automatically

Example:
SharePoint document assistant bot.

---

# What is Orchestration?

Orchestration means coordinating multiple AI services and steps together.

Example:
User Question
→ Search SharePoint
→ Send data to AI
→ Generate response
→ Return answer

This flow is orchestration.

---

# How SharePoint Can Integrate with AI

SharePoint documents can be:
- Indexed
- Searched
- Used in RAG solutions
- Sent to AI models for summarization
- Used in AI chatbots

Example:
Internal audit document chatbot.
